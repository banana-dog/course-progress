# Курс программирования - Прогресс учеников

Сайт для отслеживания прогресса учеников на курсе программирования.

## Как добавить ученика

Отредактируйте файл `data/students.json`:

```json
{
  "students": [
    {
      "id": 1,
      "name": "Имя ученика",
      "avatar": "🦊",
      "xp": 150,
      "completedLessons": [1, 2, 3],
      "specialBadges": ["🔧"]
    }
  ]
}
```

### Параметры ученика:
- `id` - уникальный номер
- `name` - имя ученика
- `avatar` - эмодзи персонажа
- `xp` - текущий опыт
- `completedLessons` - пройденные уроки (номера 1-10)
- `specialBadges` - полученные специальные бейджи (🔧, 💡, 🚀, 📚, 🌟)

## Как запустить локально

```bash
python3 -m http.server 8000
```

Откройте http://localhost:8000
